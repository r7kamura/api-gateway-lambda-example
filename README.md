# api-gateway-lambda-example
An example application to integrate Amazon API Gateway and Amazon Lambda.

## Usage
```
$ npm install
$ npm run
Scripts available in api-gateway-lambda-example via `npm run-script`:
  bundle
    gulp bundle
  clean
    gulp clean
  compile
    gulp compile
  deploy
    node script/deploy.js
  upload
    gulp upload
  zip
    gulp zip
```
